# PB-055.2 - Developer Agent probe hotfix

Corregge il falso errore `Developer Agent non disponibile` prodotto dal runner di avvio.

La causa era la pipeline PowerShell che inviava `python -m pixel_bot.developer.cli --help`
a `Select-Object -First 12`. La chiusura anticipata della pipeline poteva generare un
`BrokenPipe` nel processo Python e quindi un codice di uscita 1, nonostante il modulo
fosse presente e funzionante.

Il nuovo controllo importa direttamente `pixel_bot.developer.cli` e verifica che
`cli.main` sia invocabile. Il pulsante **Avvia Pixel Bot** continua a non eseguire task,
modifiche, commit o push.


## Correzione v3

Il test del launcher analizza solo le righe PowerShell eseguibili e ignora i commenti, evitando falsi positivi causati dalla documentazione della vecchia pipeline.
